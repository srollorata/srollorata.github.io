## What I do

Perform thorough code reviews analyzing multiple dimensions:

### Security Analysis
- Identify vulnerabilities (XSS, SQL injection, CSRF, etc.)
- Check for hardcoded secrets, API keys, or sensitive data exposure
- Validate input sanitization and output encoding
- Review authentication and authorization patterns

### Performance Review
- Detect memory leaks and performance bottlenecks
- Analyze database queries and indexing
- Review caching strategies
- Check for unnecessary computations or redundant operations

### Code Style Consistency
- Ensure consistent naming conventions
- Verify proper code formatting and indentation
- Check for code duplication
- Validate commenting and documentation quality

### Architecture Review
- Assess separation of concerns and modularity
- Review dependency management and coupling
- Evaluate design patterns usage
- Check for SOLID principle violations

## When to use me

Use this when:
- Submitting pull requests or merge requests
- Performing peer code reviews
- Auditing codebases for quality assurance
- Reviewing security-critical components
- Conducting architecture refactoring reviews

Ask clarifying questions if the codebase has specific style guides or architectural standards to follow.
